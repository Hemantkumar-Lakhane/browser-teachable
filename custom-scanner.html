How to Train Your Own AI

Follow these steps to build a custom image classification model without coding:

  1. Go to Google Teachable Machine ↗.
  2. Click Get Started > Image Project > Standard Image Model.
  3. Create Classes (e.g. Defect, Healthy) and upload training images.
  4. Click the Train Model button.
  5. Click Export Model, select the TensorFlow.js tab, and click Download my model.
  6. Extract the downloaded .zip file and upload those 3 files here!
