# Support Guide

If you need help using ModelForge AI Studio, please use the following resources:

1. **Issues**: If you found a bug or have a feature request, please [open an issue](../../issues).
2. **Discussions**: If you have a general question, please use [GitHub Discussions](../../discussions).
3. **Documentation**: Read the `docs/` folder in this repository for architecture and integration guides.

We aim to respond to issues within 48 hours.
