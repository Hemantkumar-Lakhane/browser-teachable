# Security Policy

## Supported Versions

Currently, only the latest version of **ModelForge AI Studio** is officially supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 1.0  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security is a top priority for us. If you discover any security-related issues (e.g., cross-site scripting vulnerabilities, data exposure), please do not report them via public issues.

Instead, please email the project maintainer directly or use GitHub's private vulnerability reporting feature.

1. Go to the **Security** tab of this repository.
2. Click on **Report a vulnerability**.
3. Provide a detailed summary of the issue, including steps to reproduce it and any potential impact.

We will review the report within 48 hours and work with you to test and release a patch as quickly as possible.
